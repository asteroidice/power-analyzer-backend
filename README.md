# Power Analyzer (Backend)
This is the backend component of Nathan Zimmerly and Ryan Rabello's senior project.

## Requirements
- Django v2.0

## Run The Server

1. Install Requirements

  ```
  pip install -r requirements.txt
  ```
2. Run Server
  ```
  python manage.py runserver
  ```
